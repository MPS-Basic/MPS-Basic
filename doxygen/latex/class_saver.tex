\doxysubsection{Detailed Description}
\doxylink{class_saver}{Saver} class

This class is responsible for saving the simulation results. It saves the results to the file system. It uses the \doxylink{class_particles_exporter}{Particles\+Exporter} class to export the particles to a file.
